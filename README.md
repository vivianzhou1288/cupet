# CUPet: Cornell Pet Sitting App

![CUPet Cover](cupetcover.png)

An iOS app connecting Cornell pet owners with trusted student sitters, built for Cornell's AppDev Hack Challenge Spring 2025.

**üèÜ Honorable Mention for Most Creative**

## About the Project

CUPet helps Cornell students find reliable pet care on campus through a dual-interface marketplace. Pet owners can post sitting requests with detailed pet information, dates, and location, while sitters can browse and filter opportunities by pet type, activity level, and campus area.
